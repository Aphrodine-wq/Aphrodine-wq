## Pinned Projects


### [G-Rump](https://github.com/Aphrodine-wq/G-Rump)
AI coding agent for macOS -- 54K lines of Swift, multi-model support (OpenRouter, Anthropic, OpenAI, Ollama), 100+ local tools (file ops, shell, git, system), MCP client/server, agent modes (standard, plan, parallel, speculative), Apple Intelligence integration, skills system, and ambient code awareness. Distributed via DMG with Developer ID signing and notarization.


### [AEON](https://github.com/Aphrodine-wq/AEON)
Multi-language static analysis engine -- 85K lines of Python, 20+ language adapters (Python, Java, JavaScript, TypeScript, Go, Rust, C/C++, and more), formal verification with Z3 SMT solver, algebraic effect system, and AI-native language design. Built for the age where both humans and AI write code together.


### [Driftlands](https://github.com/Aphrodine-wq/Driftlands)
Survival crafting game built with Bevy/Rust -- procedural world generation, real-time combat, crafting, farming, building, day/night cycle, weather and seasons, procedural dungeons, NPCs, save/load, and tech tree progression. Pure Rust with the Bevy 0.15 ECS engine.
